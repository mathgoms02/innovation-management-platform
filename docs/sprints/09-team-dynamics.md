# Sprint 9: Dinâmica de Equipes e Descoberta de Eventos

## Objetivo
Melhorar a jornada do Participante na formação de times e busca por hackathons.

## Tarefas Propostas

### 1. Descoberta & Busca
- [ ] **Autocomplete Search:** Refatorar busca na Home para exibir dropdown com resultados de hackathons conforme a digitação.

### 2. Gestão de Equipes (Participantes)
- [ ] **Regras de Criação:** Limitar participantes a criar apenas uma equipe como líder.
- [ ] **Fluxo de Solicitação:** Implementar sistema para solicitar entrada em equipes existentes.
- [ ] **Dashboard de Equipes:** Adicionar busca e visualização (view-only para jurados).
- [ ] **Edição de Equipe:** Permitir que o líder edite configurações básicas do time.

---

## Diretrizes Técnicas
- **CSS Grid:** Utilizar no layout do dashboard de equipes.
- **Autocomplete:** Implementar de forma performática no frontend.
